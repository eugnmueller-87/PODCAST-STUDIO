Inputs & Outputs

→ Inputs — what you feed in

  • Plain text — raw notes, bullet points, article body. Stripped and word-counted.
  • URL — any public article. Fetched via requests, cleaned with BeautifulSoup (<p> tags >40 chars).
  • PDF — uploaded file. Primary: Claude Haiku reads natively (tables, scanned pages). Fallback: PyPDF2.
  • YouTube URL — transcript fetched via youtube-transcript-api, auto-falls back to any available language.
  • Settings — podcast style, host names, target length (mins), LLM provider, temperature.

← Outputs — what you get back

  • MP3 file — stitched stereo audio, saved to outputs/episode_<timestamp>.mp3.
  • Full transcript — labelled dialogue lines (ALEX / SAM), displayed in the UI.
  • Episode metadata — auto-generated title, one-sentence summary, tags, estimated listen time.
  • Run log entry — appended to test_audio/run_log.jsonl: inputs, timing, status, script, settings.
  • Rating row — optional 1–5 star score saved to test_audio/ratings.csv with free-text notes.

Key functions — file · signature · purpose
data_processor.py
process(source, source_type) → PodcastInput

Dispatcher — routes to process_pdf, process_url, process_youtube, or process_text. Returns normalised text + word count.

data_processor.py
process_pdf(file_path) → PodcastInput

Tries Haiku native PDF first (_pdf_via_haiku). Falls back to PyPDF2 (_pdf_via_pypdf2) if key missing or call fails.

content_guard.py
check(text, context) → None | raises

Layer 1: compiled regex blocklist (instant). Layer 2: Haiku classifier at temp 0. Raises ContentViolationError on violation. Called twice — input and output.

llm_processor.py
generate_script(input, settings, temperature) → PodcastScript

Builds style prompt, calls Anthropic or OpenAI based on settings.llm_provider, parses raw response into DialogueLine list + EpisodeMetadata, then runs humanizer.

humanizer.py
humanize_script(script) → PodcastScript

Iterates every line. Probabilistically injects reactions (45%), openers (25%), self-corrections (15%), Sam quips (30%), Sam self-deprecation (15%).

tts_generator.py
synthesise_script(script, settings, provider) → AudioOutput

Splits lines at — / …, computes prosody params, calls ElevenLabs or edge-tts in parallel (asyncio.gather), stitches segments with dynamic silences via pydub, exports MP3.

Environment variables
Required

ANTHROPIC_API_KEY

Powers Claude Sonnet (script), Haiku (PDF + safety guard). Without it the app will not start.

Optional — better voices

ELEVENLABS_API_KEY

Enables ElevenLabs eleven_v3 for Alex. If missing, Alex falls back to en-GB-SoniaNeural via edge-tts.

Optional — alternative LLM

OPENAI_API_KEY

Enables GPT-4o as the script generator. Selectable in the UI via the LLM Provider toggle.
