What it does

One tool. Any content in. Full podcast out.

PodcastIQ is an automated podcast studio focused on AI startups, work, and society. Feed it any source — it returns a two-host dialogue script, neural audio, episode metadata, and a downloadable MP3.

Core features
Script Generation

Dual LLM — Claude Sonnet 4.6 or GPT-4o

Four styles: Educational, Debate, News Brief, Deep Dive. Each has its own prompt template with host personas and length targets.

Voice Synthesis

ElevenLabs (Alex) + edge-tts (Sam)

Alex: ElevenLabs eleven_v3, premium neural voice. Sam: Microsoft en-US-GuyNeural, free, always available. Parallel synthesis via asyncio.

Humanizer

3-layer realism post-processor

Reactions (45%), openers (25%), self-corrections (15%), Sam dry humour quips (30%) and self-deprecation (15%) — applied after the LLM.

Emotional Prosody

Pitch + rate adapt per line

Questions rise in pitch (−8% rate, +10 Hz). Excitement speeds up. Reflection slows down. Em-dashes → 420 ms silence. Ellipsis → 700 ms.

Content Safety

2-layer guard — input AND output

Layer 1: instant regex blocklist. Layer 2: Claude Haiku classifier (temp 0). Blocks racism, hate speech, foul language at both checkpoints.

Logging + Rating

Full run transparency

Every run logged to run_log.jsonl (inputs, timing, script, status). In-app 1–5 star rating saved to ratings.csv. Visible on GitHub.

What's shipped
  • Multi-source ingestion (PDF, URL, YouTube, text)
  • Dual LLM — Claude Sonnet 4.6 + GPT-4o in UI
  • ElevenLabs premium voice + edge-tts fallback
  • Emotional prosody — pitch & rate per line
  • 3-layer humanizer + Sam dry humour
  • Parallel TTS synthesis (asyncio.gather)
  • Content safety guard (regex + Haiku, 2 checkpoints)
  • Episode metadata — title, summary, tags, duration
  • Run log (JSONL) + ratings log (CSV)
  • Rating UI — 1–5 stars, transcript + audio quality
  • Temperature slider per LLM provider
  • Dark studio Gradio UI, share=True live URL
