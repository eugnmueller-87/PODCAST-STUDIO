Humanizer Layer

Making the dialogue feel genuinely human

Reactions

45%

"Hmm." / "Right." / "Wait —" prepended based on what the previous line said.

Openers

25%

"You know," / "Here's the thing —" / "Okay so" injected at line start.

Self-corrections

15%

Mid-sentence: "— or, actually, —" / "— scratch that —" for authenticity.

Sam's Dry Humour

30%

"Classic Silicon Valley — disrupt first, apologise later." Appended to Sam's lines only.

Sam also gets self-deprecation (15%): "— don't fact-check me on that, I'm going from memory."

Safety & Logging

Content guard + full run transparency

🛡

2-Layer Content Guard

Applied before the LLM and after humanization.

  • ✔
    Layer 1 — Regex blocklist (instant, no API call): slurs, hate phrases, calls for violence.
  • ✔
    Layer 2 — Claude Haiku classifier (temp 0, ~60 tokens): coded language, implied discrimination, subtle abuse.
  • ✔
    Violations show 🚫 Blocked for Harmful Content in the UI and are logged as "status": "blocked".
📋
